Differentiable Nonparametric Belief Propagation for Articulated Pose Tracking

First Author, Second Author, Third Author...

We present a differentiable approach to learn the probabilistic factors used for inference by the Pull Message Passing Nonparametric Belief Propagation (PMPNBP) algorithm. Existing nonparametric belief propagation methods rely on domain-specific features encoded in the probabilistic factors of a graphical model. In this work, we replace each hand crafted factor with a differentiable neural network enabling the factors to be learned using an efficient optimization routine from labeled data. By combining differentiable neural networks with an efficient belief propagation algorithm our method is able to maintain a set of marginal posterior samples while simultaneously learning the factors needed for articulated pose tracking using end-to-end training. We evaluate our differentiable nonparametric belief propagation (DNBP) method on a set of articulated tracking tasks and compare performance with a recurrent neural network. Results from this comparison demonstrate the effectiveness of using learned factors for tracking and suggest the practical advantage over hand-crafted approaches.

Experiments

DNBP tracks the belief of each node of a graph model over a sequence of input data. The edge parameters and observation likelihood are learned. We show selected results on two different graph models.
